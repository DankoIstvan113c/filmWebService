REST API webszolgáltatásai

Github repository: https://github.com/DankoIstvan113c/filmWebService/tree/master

Metódus URL Szolgáltatás
GET /films Filmek lisjáta
GET /categories/{id} Egy film
GET /films/{id}/szineszek Egy film szereplői
POST /films Új film létrehozása
DELETE /films/{id} Film törlése
PATCH /films/{id}/{hossz} Film módosítása (hossz)
GET /szineszek Minden színész
GET /szineszek/{id} Egy színész
POST /szineszek Új színész felvitele
PATCH /szineszek/{id}/{name}/{age} Színész módosítása (név, kor)
PUT /szineszek/{id} Színész módosítása (név, kor)
DELETE /szineszek/{id} Színész törlése
